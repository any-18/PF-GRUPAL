# Ola Violeta 
Es una aplicación web full-stack, de geolocalización desarrollada con tecnologías como:
- `ReactJS` 
- `Redux` 
- `CSS3-Modules` 
- `Node.js`
- `Express`
- `Sequelize` 
- `PostgreSQL` 
- `Google Maps API`
